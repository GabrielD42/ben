4 This repository contains code that will (hopefully) one day become a robot. BEN stands for **B**right **E**nough to **N**avigate. BEN has a multi-step goal:
5 
6 1. Navigate to a sector in his world. At this point he does not know where any obstacles are
7 2. Search the sector for a small object, identified by comparing it to pictures it has taken
8 3. Pick up the object and return to his starting position on the shortest path
