Detailed Description

Provides a set of colors for console output.

This is only known to work on Linux. These colors should be used by passing the enum values to the standard stream.

Ex:

cout << Color::GREEN << "this is green!" << Color::DEFAULT << endl;

make sure to reset the color every time.
