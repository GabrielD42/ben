Detailed Description

a class for testing code

This class has very simple usage. For each test you would like to perform, create a new Test object. Ideally this should be done inside a free function. Run any number of assertions on it using its built in methods. Test will log the success of each of these assertions. Upon destruction, it will log if tests were successful
